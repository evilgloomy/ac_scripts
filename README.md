# AC-Scripts

Custom Shaders Patch (CSP) Lua scripts for Assetto Corsa.

## Contents

| Script | Description |
|---|---|
| [`cars/koenigsegg_cc850`](cars/koenigsegg_cc850) | Koenigsegg CC850 Engage Shift System (ESS): a virtual 6-speed manual layered over the 9-speed Light Speed Transmission, with Normal/Track drive modes and a full AUTO (D) mode. |

## Requirements

- Assetto Corsa with Custom Shaders Patch **0.3.0-preview210 or newer**
- Content Manager recommended

Each car folder has its own README with installation and configuration details.
